=head1 名前

Acme::Bushisms - ブッツュ氏大いにPerlを語る (Dubya Does Perl)

=head1 概要

	use Acme::Bushisms;

	print "Hello world";

=head1 説明

The first time you run a program under C<use Acme::Bushisms>, the module
removes all the unsightly printable, democrat, and liberal characters from 
your source file. The code continues to work exactly as it did before, 
but now it contains Bush speak:

最初にC<use Acme::Bushisms>の下でプログラムを実行すると、このモジュールは
あなたのソースファイルから、あらゆる目障りな出版可能で、民主党的で
リベラルな性格（文字）を取り除いてくれる。コードは以前と全く同様に
動きつづける。しかし、今やそこにはブッシュのおしゃべりが含まれている。

      	use Acme::Bushisms;

	"Families is where our nation finds hope, where wings take dream."
	--George W Bush, LaCrosse, Wis., Oct. 18, 2000
        

    [家族とわ我が国が希望を見出し、羽ばたくが夢の所なのです]

=head1 注意

The Bushisms are random quotes from the Complete Bushisms Website by 
Jacob Weisberg on Slate. The address is http://slate.msn.com/id/76886
If this page is unreachable, then a default Bushism is used. 

このブッシズムは Slate の Jacob Weisberg がやっている
Complete Bushisms Website からランダムに飲尿している。
アドレスは http://slate.msn.com/id/76886 。
このページにアクセスできない場合は、デフォルトのブッシズムが使われる。

[デフォルト："I know the human being and fish can coexist peacefully."
：人類とお魚とは平和的に共存できます]
